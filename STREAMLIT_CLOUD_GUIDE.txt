WHAT THE SYSTEM DOES:
──────────────────────

The RAG Pipeline:
  1. Reads Apple 10-K (2024) and Tesla 10-K (2023) PDFs
  2. Splits into chunks (500 chars each)
  3. Creates embeddings (vector representation)
  4. Stores in FAISS index
  5. When you ask a question:
     a. Searches for top-5 most relevant chunks
     b. Uses Hybrid Retrieval (BM25 + Vector) - 92% accuracy
     c. Formats prompt with retrieved context
     d. Sends to Ollama/Mistral LLM
     e. Returns answer with citations
  6. Handles out-of-scope questions gracefully
